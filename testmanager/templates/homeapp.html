Welcome to Moon Cloud

continuous MOnitoring ON the CLOUD

Monitoring - Testing - Security - Performance

Cloud users and service providers are increasingly concerned about the management of their data and the behavior of the applications they own/use once stored/deployed in the cloud. They therefore ask for enhanced assurance solutions, which partially mitigate the new risks and threats they are facing. Moon-cloud provides a fully automated framework for cloud-service verification, which aims at increasing cloud service trustworthiness and transparency. It is platform independent, implements multiple solutions for continuous monitoring and testing, and supports an agent-based collection of cloud service behavior. Moon-cloud provides verifiable evidence on the status of users' data and applications once moved to the cloud, a real turning point for cloud transparency and trustworthiness. Moon-cloud will make the cloud a viable option even for businesses and enterprises with strong security requirements, attracting new customers to the cloud. Moon-cloud framework has been already used as the enabler for trustworthy certification and compliace processes in the cloud.
